**For running the project, execute these two commands:**

1. **🏗 Download all dependencies**
   ```
   cd client && npm i && cd ../server && npm i
   ```
2. **🏁 Run project**
   ```
   npm start
   ```

---

**⚙ `.env` variables**

```
# Server
PORT = 8080
DOMAIN  = http://localhost:8080

# MySQL
```
